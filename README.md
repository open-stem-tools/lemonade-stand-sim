# Lemonade Stand Simulator

An educational web app that teaches business math, pricing strategy, and microeconomics through running a lemonade stand.

Designed for students ages 8-12.

## Topics Covered

- Recipe ratios and unit conversions
- Cost-per-unit calculations
- Bulk purchasing (rounding up)
- Pricing strategy and profit margins
- Loan interest and break-even analysis
- Supply/demand and weather-based decision making
- Production planning and waste management

## Usage

Open `index.html` in any browser. No build step, no dependencies, no internet required.

Teacher settings (store prices, base customer count) can be configured at `settings.html`.
